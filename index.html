Guia de utilização

O que faz cada separador

📊 Resumo — visão geral do crédito: capital em dívida, prestação actual, juros pagos e evolução gráfica do capital.
📈 Euribor & Cenários — regista revisões reais da Euribor e define três cenários futuros (optimista, base, pessimista).
📋 Plano — tabela completa de amortização mês a mês, com destaque para o mês actual.
💰 Abates — regista abates antecipados realizados e simula o impacto de futuros abates no custo total.
🧾 Custos — regista despesas recorrentes como seguros de vida, multiriscos e taxas de manutenção.
⚙️ Configuração — dados do contrato, data de início, exportação e importação de dados.

Onde estão os dados

Todos os dados são guardados exclusivamente no teu browser (localStorage). Não são enviados para nenhum servidor. Se limpares o cache ou os dados do browser, os dados são apagados.

💡 Para não perderes os dados, usa a função Exportar em Configuração → Exportar / Importar dados. Guarda o ficheiro .json num local seguro.

Exportar e importar dados

Em Configuração, clica em ⬇ Exportar dados para guardar tudo num ficheiro JSON. Para restaurar, clica em ⬆ Importar dados e selecciona o ficheiro. Útil para mudar de browser ou fazer backup.

Notas sobre os cálculos

O plano usa o método de amortização francês (prestação constante). Em períodos de taxa variável, a prestação é recalculada com base em: Euribor + spread, capital em dívida e meses restantes. Os cenários futuros são estimativas.

Sobre o projecto

Uma ferramenta gratuita e open-source para simular e acompanhar empréstimos à habitação.

  • Simulação de plano de pagamentos com taxa fixa e variável
  • Histórico de revisões de taxa Euribor
  • Cenários optimista, base e pessimista para previsões
  • Simulação de amortizações antecipadas
  • Dados guardados automaticamente no browser
